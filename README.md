# kosekose

ブラウザで遊べるシンプルなkosekoseゲームです。

## 遊び方

1. `index.html` をブラウザで開く
2. スペースキーでゲーム開始
3. 同じ色のぷよを4つ以上つなげて消す
4. 連鎖でハイスコアを狙おう！

## 操作方法

- **← →** : 左右移動
- **↓** : 高速落下
- **Z** : 左回転
- **X** : 右回転
- **スペース** : ゲーム開始/一時停止

## ゲームルール

- フィールドは6×12マス
- 2つ1組のぷよが落下
- 同じ色を4つ以上つなげると消える
- 連鎖するとスコアがアップ
- フィールドの一番上までぷよが積み上がるとゲームオーバー

## カスタム画像の使用方法

`img` ディレクトリに以下のファイル名で画像を配置すると、自動的にぷよの画像として使用されます：

- `001.png` : 赤ぷよ
- `002.png` : 青ぷよ
- `003.png` : 緑ぷよ
- `004.png` : 黄ぷよ

画像がない場合は、デフォルトの円形ぷよが表示されます。

## BGMの設定方法

`audio` ディレクトリに `bgm.mp3` ファイルを配置すると、BGMとして再生されます。

- BGM設定パネルでON/OFF切り替え可能
- 音量調整スライダーで音量調整可能
- 設定はブラウザに保存されます

## 連鎖効果音の設定方法

`audio` ディレクトリに以下のファイルを配置すると、連鎖時に効果音が再生されます：

- `chain1.mp3` : 2-3連鎖用
- `chain2.mp3` : 4-6連鎖用  
- `chain3.mp3` : 7連鎖以上用

効果音の音量はBGMの音量スライダーと連動します。
